# jlin80.github.io — Personal Portfolio

Personal portfolio website built from scratch with HTML5, CSS, and 
vanilla JavaScript.

## Features
- Retro terminal / cyberpunk aesthetic
- Bilingual — English / Spanish (custom i18n, no libraries)
- Dark / light mode with localStorage persistence
- ASCII rain, glitch effects, scroll-reveal animations
- Live clock and typewriter hero effect
- Responsive design
- Functional contact form

## Stack
HTML5 · CSS3 · JavaScript (vanilla) · GitHub Pages
